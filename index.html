🎵 Spotify Data Analytics

Analyze, visualize, and explore your favorite tracks

🧾 Key Features

  • ✔ Extract metadata from Spotify tracks
  • ✔ Store data in MySQL
  • ✔ Visualize insights (optional with Python)

📂 Project Files

  • spotify.py: Extract data from Spotify
  • spotify_mysql.py: Store data into MySQL
  • spotify_mysql_urls.py: Process multiple track URLs
  • spotify.sql: MySQL schema
  • track_urls.txt: Track URLs list

🧑‍💻 How to Run

  1. Install dependencies: pip install spotipy mysql-connector-python pandas
  2. Run spotify_mysql_urls.py with track_urls.txt file
